## 🤝 Contributing

Community contributions are welcome and encouraged! Here's how you can help keep this archive up to date.

### Ways to contribute

**Option 1 — Open an Issue**
If a hack is missing, broken, or a newer version exists, [open an issue](../../issues/new) and describe it. Include the hack name, version, and a source link if possible. Someone will pick it up.

**Option 2 — Fork & Pull Request**
If you have the file ready, fork the repo, add it yourself, and open a PR.

---

### 📋 Contribution Rules

1. **File format:** Always submit a **`.gba` file** — the full, pre-patched ROM. Do **not** submit `.ips`, `.ups`, or any raw patch file. Others should be able to play it immediately without extra steps.
   - `.gb` and `.gbc` files are also accepted if the hack targets those platforms.

2. **Never work directly on `main`.** Always create a new branch from `main`, make your changes there, and open a PR targeting `main`. Example:
   ```
   git checkout -b add/pokemon-insurgence-gba
   ```

3. **One hack per PR** where possible. Keeps history clean and reviews fast.

4. **Name your file clearly.** Use the format: `Pokemon HackName vX.X.gba`

5. **No duplicates.** Check existing files before adding. If you have a newer version of an existing hack, replace the old file and note the version change in the PR description.

6. **No malicious files.** All submissions may be reviewed. Anything that isn't a legitimate ROM hack will be rejected.

## ⚙️ How to Play

1. Download any `.gba` file from this repo
2. Open it with an emulator:
   - **[mGBA](https://mgba.io/)** — recommended, most accurate
   - **VisualBoyAdvance-M** — widely supported
   - **RetroArch** (mGBA or VBA-M core)
3. Or flash to a GBA flash cartridge (EZ-Flash, Everdrive) for real hardware play
